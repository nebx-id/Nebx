# Nebx Building Blocks for .NET

Reusable, high-quality .NET libraries designed to simplify common application patterns, reduce boilerplate, and accelerate development.  
All packages follow a consistent structure, reliable versioning, and strict code quality standards.

Nebx.BuildingBlocks provides modular libraries you can use independently or together.
